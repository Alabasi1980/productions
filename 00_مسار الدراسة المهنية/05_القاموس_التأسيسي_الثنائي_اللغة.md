# القاموس التأسيسي الثنائي اللغة

## الهدف من هذا الملف

هذا الملف ليس قاموسًا نهائيًا لكل الكتاب، بل Foundation Bilingual Glossary.

استخدمه يوميًا في البداية حتى تصبح المصطلحات الأساسية مألوفة لك بالعربية والإنجليزية معًا.

## مصطلحات الجوهر العام

| العربية | English | المعنى المختصر |
| --- | --- | --- |
| الإنتاج | Production | عملية تحويل المواد إلى منتج نهائي أو نصف مصنع |
| التصنيع | Manufacturing | الإطار الأوسع لعمليات الإنتاج الصناعية |
| موديول الإنتاج | Production Module | الجزء المسؤول عن دورة الإنتاج داخل ERP |
| البيانات الأساسية | Master Data | البيانات الثابتة التي تقوم عليها دورة الإنتاج |
| المنتج النهائي | Finished Good | المنتج الذي يخرج من أمر الإنتاج كنتاج نهائي |
| مادة خام | Raw Material | مادة تدخل في تصنيع المنتج |
| منتج نصف مصنع | Semi-finished Good / Subassembly | منتج وسيط قد يكون Input لأمر آخر |
| وحدة القياس | Unit of Measure (UoM) | الوحدة المستخدمة للكميات مثل قطعة أو كجم |
| مستودع | Warehouse | المكان المنطقي أو الفعلي لتخزين المواد |
| موقع | Location / Locator | موضع داخلي أدق من المستودع |

## مصطلحات تصميم المنتج والتصنيع

| العربية | English | المعنى المختصر |
| --- | --- | --- |
| قائمة المواد | Bill of Material (BOM) | قائمة المكونات والكميات المطلوبة لتصنيع المنتج |
| وصفة | Formula | بنية مكونات تستخدم غالبًا في التصنيع العملياتي |
| هيكل الصنف | Item Structure | مصطلح Oracle للبنية المادية للمنتج |
| مسار التصنيع | Route / Routing | الخطوات التي يمر بها المنتج أثناء التصنيع |
| عملية | Operation | خطوة تشغيلية داخل Route أو Work Definition |
| مورد | Resource | عامل أو آلة أو أداة أو مورد خارجي يدخل في التنفيذ |
| مركز عمل | Work Center | مكان أو وحدة تنفيذ للعمليات |
| تعريف العمل | Work Definition | قالب Oracle الذي يجمع المواد والعمليات والموارد |
| هالك | Scrap | فاقد في المواد أو المنتج أو العملية |
| منتج جانبي | By-product | مخرج ثانوي ينتج مع المنتج الرئيسي |
| منتج مشترك | Co-product | مخرج رئيسي إضافي يشارك المنتج الأساسي في التكلفة |

## مصطلحات التنفيذ

| العربية | English | المعنى المختصر |
| --- | --- | --- |
| أمر إنتاج | Production Order | أمر تنفيذ إنتاج في كثير من الأنظمة |
| أمر عمل | Work Order | المصطلح التنفيذي الشائع خاصة في Oracle |
| إنشاء | Created | حالة إنشاء الأمر قبل التنفيذ |
| إطلاق | Released | حالة يصبح فيها الأمر جاهزًا للتنفيذ |
| بدء | Started | بدء التنفيذ الفعلي للأمر |
| صرف مواد | Material Issue / Goods Issue | إخراج المواد من المخزون إلى الإنتاج |
| تسجيل تشغيل | Operation Execution / Reporting | تسجيل العمل أو الوقت أو التقدم |
| استلام منتج نهائي | Finished Goods Receipt | دخول المنتج النهائي إلى المخزون |
| إكمال المنتج | Product Completion | مصطلح Oracle لاستلام المنتج الناتج |
| الإبلاغ كمنجز | Report as Finished | مصطلح Dynamics لإثبات إنجاز المنتج |
| إرجاع مواد | Material Return | إعادة مواد من الأمر إلى المخزون |
| إغلاق | Close / Closing | حسم الأمر تشغيليًا وماليًا بحسب النظام |

## مصطلحات التكلفة والرقابة

| العربية | English | المعنى المختصر |
| --- | --- | --- |
| تكلفة مقدرة | Estimated Cost | تكلفة محسوبة قبل أو أثناء التنفيذ كمؤشر متوقع |
| تكلفة معيارية | Standard Cost | تكلفة مرجعية ثابتة أو معتمدة للمقارنة |
| تكلفة فعلية | Actual Cost | التكلفة الحقيقية المبنية على الحركات الفعلية |
| تكلفة تحت التشغيل | Work in Process (WIP) | قيمة ما صُرف أو سُجل على أمر لم يكتمل إغلاقه |
| انحراف | Variance | الفرق بين المتوقع والمعياري والفعلي |
| تحميل تكلفة | Cost Posting / Cost Charging | ربط حدث تشغيلي بأثر تكلفة |
| توزيع تكلفة | Cost Distribution | مصطلح Oracle لتحليل وربط التكلفة بالحركات |
| إغلاق الفترة | Period Close | إنهاء الفترة المالية بعد مراجعة الحركات والتكلفة |
| تقارير رقابية | Control Reports | تقارير لمتابعة الانحرافات والأوامر المفتوحة وWIP |

## مصطلحات الجودة والتتبع

| العربية | English | المعنى المختصر |
| --- | --- | --- |
| جودة | Quality | التحقق من مطابقة المنتج أو العملية للمطلوب |
| أمر جودة | Quality Order | كيان رقابي في بعض الأنظمة لفحص الجودة |
| مشكلة جودة | Quality Issue | حالة انحراف أو خلل يحتاج متابعة |
| دفعة | Lot / Batch | مجموعة متتبعة من مادة أو منتج |
| رقم تسلسلي | Serial Number | معرف منفرد لوحدة محددة |
| تتبع | Traceability | القدرة على تتبع المادة أو المنتج عبر الدورة |
| إعادة تشغيل | Rework | إعادة معالجة منتج أو كمية غير مطابقة |
| رفض | Reject | كمية مرفوضة لا تقبل كمنتج جيد |

## مصطلحات استشارية مهمة

| العربية | English | المعنى المختصر |
| --- | --- | --- |
| تحليل الفجوات | Gap Analysis | مقارنة النظام الحالي بالمرجع المطلوب |
| متطلب وظيفي | Functional Requirement | ما يجب أن يحققه النظام من منظور العمل |
| تصميم وظيفي إلى تقني | Functional to Technical Design | ترجمة المنطق الوظيفي إلى منطق قابل للتطوير |

| اختبار قبول المستخدم | User Acceptance Testing (UAT) | اختبار الدورة الحقيقية قبل التشغيل الرسمي |
| تشغيل فعلي | Go-Live | بداية استخدام النظام في الواقع |

## قاعدة دراسة هذا الملف

لا تحفظه حفظًا صامتًا.

بل حاول في كل مصطلح أن تجيب عن ثلاثة أسئلة:

1. ما معناه؟
2. أين يدخل في الدورة؟
3. ما أثره إذا كان غائبًا أو ضعيفًا؟
